# 🚀 AI Resume Builder

An AI-powered Full-Stack Resume Builder built with the MERN Stack that helps users create, customize, and share professional resumes effortlessly. The application leverages **Google Gemini AI** to enhance resume content and **ImageKit AI** for intelligent profile image processing, delivering a modern and user-friendly resume-building experience.

---

## ✨ Features

- 🔐 Secure User Authentication (JWT)
- 🤖 AI-Powered Professional Summary Enhancement
- 📝 AI-Enhanced Job Descriptions
- 📄 Create & Manage Multiple Resumes
- 📤 Upload Existing Resume & Auto-Fill Data
- 👀 Real-Time Resume Preview
- 🎨 Multiple Professional Resume Templates
- 🌈 Custom Theme & Accent Colors
- 🖼️ AI Background Removal for Profile Images
- 📥 Download Resume as PDF
- 🔗 Public & Private Resume Sharing
- 🌐 Shareable Live Resume Links
- 📱 Fully Responsive Design
